# 사진 해시태그 추천 웹사이트 프로젝트 - Waffle

이 프로젝트는 사용자가 사진을 업로드하면 적절한 해시태그를 추천해주는 웹사이트 개발 프로젝트입니다.

## 핵심 기능

사진을 업로드하면 AI 기반으로 적절한 해시태그를 추천해주며, 추천 받은 해시태그를 이용해 사진을 소셜 미디어에 직접 공유할 수 있습니다.
